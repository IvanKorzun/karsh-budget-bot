# 🚗 Karsh Budget Bot

Бот-бухгалтер для совместных поездок на каршеринге.

## Фишки:
- Интерактивные опросы `/poll` со списком участников.
- Умный подхват экипажа по юзернеймам.
- Управление балансами (депозиты и долги).
- Реакции 🔥 и 🤡 для группового чата.
- Упакован в Docker (multi-stage build).

## Как запустить:
1. Переименуй `.env.example` в `.env` и вставь свой токен.
2. Убедись, что у тебя есть `docker-compose`.
3. Запусти: `docker-compose up -d --build`
